# Typing Speed Test Website ⌨️⚡

A modern and interactive typing speed test website built using **HTML, CSS, and JavaScript**.
This project helps users measure their typing speed, accuracy, and reaction time in real-time while practicing keyboard skills.

The application dynamically tracks user input, calculates **Words Per Minute (WPM)**, counts mistakes, and provides a smooth typing experience with a clean user interface.

# ⚙️ How the Project Works

The typing test starts when the user begins typing the displayed paragraph.

The JavaScript logic continuously compares:

* The original text
* The user typed text

The application then:

* Detects correct and incorrect characters
* Updates mistakes count
* Calculates typing speed in real time
* Tracks elapsed time
* Displays WPM and CPM statistics

When the timer ends, the test automatically stops and shows the final results.
